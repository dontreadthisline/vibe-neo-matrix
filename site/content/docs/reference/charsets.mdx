---
title: 字符源
description: neo-rainst 支持的字符集和自定义字符源
---

neo-rainst 支持多种字符来源，通过 `--charset` 参数或配置文件中的 `charset.source` 指定。

## 内置字符集

| 名称 | 别名 | Unicode 范围 | 说明 |
|------|------|-------------|------|
| `katakana` | — | `U+FF64`–`U+FF9F` | 半角片假名（默认） |
| `ascii` | `default` | `A-Z a-z 0-9` | ASCII 字母和数字 |
| `extended` | `extended_default` | ASCII + 半角片假名 | 扩展默认集 |
| `english` | `letters` | `A-Z a-z` | 仅英文字母 |
| `digits` | `dec`, `decimal` | `0-9` | 仅数字 |
| `punc` | `punctuation` | 标点符号范围 | 仅标点 |
| `binary` | `bin` | `0-1` | 二进制 |
| `hex` | `hexadecimal` | `0-9 A-F` | 十六进制 |
| `greek` | — | `U+0370`–`U+03FF` | 希腊字母 |
| `cyrillic` | — | `U+0410`–`U+044F` | 西里尔字母 |
| `arabic` | — | `U+0627`–`U+0649` | 阿拉伯字母 |
| `hebrew` | — | `U+0590`–`U+05FF` | 希伯来字母 |
| `devanagari` | — | `U+0900`–`U+097F` | 天城文 |
| `braille` | — | `U+2800`–`U+28FF` | 盲文 |
| `runic` | — | `U+16A0`–`U+16FF` | 卢恩符文 |

## 自定义字符源

### 从文件加载

```bash
neo-rainst --charset-file ~/my-chars.txt
```

或在配置文件中：

```toml
[charset]
source = "file:/home/user/my-chars.txt"
```

### 从 stdin 加载

```bash
echo "Hello World" | neo-rainst --charset-stdin
```

### Unicode 码点范围

```bash
# 平假名范围
neo-rainst --chars "0x3040,0x309F"
```

## Claude Code 会话

```bash
neo-rainst --charset claude
```

自动从 Claude Code transcript 目录加载最近对话内容。详见 [Claude Code 集成](/vibe-neo-matrix/claude-integration/)。
