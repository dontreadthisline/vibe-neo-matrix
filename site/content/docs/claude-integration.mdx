---
title: Claude Code 集成
description: 将 Claude Code 对话渲染为 Matrix 数字雨
---

neo-rainst 可以自动提取 Claude Code 的对话记录，将你和 AI 的交流内容渲染成数字雨效果。

## 工作原理

1. Claude Code 在执行 `/exit` 或退出时将对话内容写入 `~/.claude/projects/<project-dir>/` 下的 JSONL 文件
2. neo-rainst 在 `claude` 模式下扫描该目录，找到最近修改的 transcript 文件
3. 提取 user 和 assistant 消息中的纯文本作为字符源
4. 将文本内容渲染为数字雨，每 ~1s 检查一次新内容

## 配置方式

### 方式一：CLI 参数

```bash
neo-rainst --charset claude --exit-on-key
```

### 方式二：配置文件

在 `.neo-rainst.toml` 或 `~/.config/neo-rainst/config.toml` 中：

```toml
[charset]
source = "claude"

[claude]
enabled = true
transcript_dir = ""   # 留空则根据 CWD 自动推导
max_chars = 10000

[exit]
mode = "on-key"
```

## SessionEnd Hook 集成

最简单的使用方式是通过 Claude Code 的 SessionEnd hook 自动触发：

### 1. 复制 hook 脚本

```bash
cp scripts/session-end.py .claude/hooks/
chmod +x .claude/hooks/session-end.py
```

### 2. 配置 `.claude/settings.json`

```json
{
  "hooks": {
    "SessionEnd": [
      {
        "matcher": "*",
        "hooks": [
          {
            "type": "command",
            "command": "python3 .claude/hooks/session-end.py",
            "timeout": 30
          }
        ]
      }
    ]
  }
}
```

### 3. 使用流程

1. 正常使用 Claude Code 进行对话
2. 输入 `/exit` 结束会话
3. SessionEnd hook 自动提取会话文字并启动 neo-rainst
4. 数字雨展示当前会话内容，按任意键退出

## bashrc 快捷函数

在 `~/.bashrc` 中添加：

```bash
c() {
  cc_ds
  claude --model deepseek-v4-pro --dangerously-skip-permissions --allowedTools all "$@"
  neo-rainst --exit-on-key
}
```

每次 Claude Code 会话结束后，自动渲染数字雨。
