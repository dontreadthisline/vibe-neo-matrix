neo-rainst 配置文件格式和优先级

neo-rainst 支持三级配置覆盖，优先级从高到低：

  1. CLI 参数 — 命令行直接指定
  2. 项目级配置 — 当前目录下的 .neo-rainst.toml
  3. XDG 用户配置 — ~/.config/neo-rainst/config.toml

[charset]

Section titled “[charset]”
字段类型默认值说明
sourcestring"katakana"字符源名称，或 file:/path/to/file 加载文件

[render]

Section titled “[render]”
字段类型默认值说明
fpsfloat60.0帧率
colorstring"green"颜色主题
shading_modeint0着色模式: 0=随机, 1=距离头部渐变
bold_modeint1粗体模式: 0=关闭, 1=随机, 2=全部
full_widthboolfalse全宽字符模式
default_bgbooltrue使用终端默认背景色
show_statusboolfalse显示底部状态栏

[rain]

Section titled “[rain]”
字段类型默认值说明
speedfloat8.0下落速度（字符/秒）
densityfloat1.0雨滴密度
async_scrollboolfalse异步滚动
max_droplets_per_colint3每列最大雨滴数
short_pctfloat50.0短雨滴百分比
die_early_pctfloat33.3提前消失百分比

[glitch]

Section titled “[glitch]”
字段类型默认值说明
enabledbooltrue启用故障效果
pctfloat10.0故障百分比
low_msint300最小故障持续时间
high_msint400最大故障持续时间

[exit]

Section titled “[exit]”
字段类型默认值说明
modestring"normal"退出模式: normal / on-key / after-secs
secsfloat0.0mode=after-secs 时的秒数

[claude]

Section titled “[claude]”
字段类型默认值说明
enabledboolfalse启用 Claude Code 字符源
transcript_dirstring""Transcript 目录，留空自动推导
max_charsint10000最大加载字符数
