快速开始

安装并运行 neo-rainst 终端数字雨

neo-rainst 是一个用 Rust 编写的终端 Matrix 数字雨模拟器，基于 ratatui 和 crossterm。

Section titled “键盘控制”
按键功能
q / Esc退出
空格重置数字雨
p暂停 / 继续
↑ / ↓加速 / 减速
← / →减少 / 增加故障效果比例
Tab切换着色模式
a切换异步滚动
- / +减少 / 增加雨滴密度
1 - 0切换颜色主题
! - %切换更多颜色主题
