neo-rainst

终端里的 Matrix 数字雨。用 Rust + ratatui 重写，支持自定义字符源和 Claude Code 会话渲染。

快速开始 GitHub

一行命令，终端变身 Matrix 屏幕

用片假名、ASCII、二进制或你的 Claude Code 对话记录，在终端中渲染经典的绿色数字雨。

安装
cargo install --git https://github.com/dontreadthisline/vibe-neo-matrix.git
运行
neo-rainst --charset katakana --color green \
  --speed 8.0 --density 1.0 \
  --message "Wake up, Neo..."

为什么选择 neo-rainst

多字符源

内置 katakana、ascii、binary、hex、greek、cyrillic、braille 等 13+ 种字符集，支持从文件或 stdin 加载自定义字符。

Claude Code 集成

自动提取 Claude Code 对话记录，将你和 AI 的交流实时渲染成数字雨。SessionEnd hook 触发，无缝衔接。

16 种颜色主题

从经典绿色到 rainbow、vaporwave、gold、pink，支持 truecolor / 256 色 / 16 色自动检测。

交互式控制

空格重置、p 暂停、方向键调速、Tab 切换着色模式、数字键切换颜色主题。一切尽在键盘。

灵活配置

CLI 参数、项目级 .neo-rainst.toml、XDG 用户配置三级覆盖。精确控制速度、密度、故障效果、渐隐时间。

多种运行模式

全屏模式、内联模式、屏保模式、定时退出、任意键退出。适配演示、屏保、终端美化等场景。

消息揭示效果

输入 --message "Wake up, Neo..." 让数字雨如电影般揭示隐藏文字，营造经典 Matrix 氛围。

轻量高性能

Rust 编写，基于 ratatui 和 crossterm。极低资源占用，流畅 60fps 动画，精准帧率控制。

开始使用

通过 Cargo 安装

cargo install --git https://github.com/dontreadthisline/vibe-neo-matrix.git

或使用一键安装脚本

curl -fsSL https://raw.githubusercontent.com/dontreadthisline/vibe-neo-matrix/master/install.sh | bash
